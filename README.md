# Pasos de instalacion

1. Ejecutar el comando `composer instal`

2. Renombrar el archivo `.env.example` a `.env` y reemplazar los valores

3. Ejecutar migraciones.

```bash
php artisan migrate
```

4. Correr el proyecto.

```bash
 php artiisan serve
```
